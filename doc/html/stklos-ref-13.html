Aubrey Jaffer maintains a package called SLIB which is a portable Scheme library which provides compatibility and utility functions for all standard Scheme implementations. To use this package, you have just to type

(require "slib")

and follow the instructions given in the SLIB library to use a particular package.

Note: SLIB uses also the require and provide mechanism to load components of the library. Once SLIB has been loaded, the standard STklos require and provide are overloaded such as if their parameter is a string this is the old STklos procedure which is called, and if their parameter is a symbol, this is the SLIB one which is called.
